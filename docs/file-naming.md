# File Naming

Use predictable names so cohosts and Codex can find things quickly.

## Episodes

Folder pattern:

```text
episodes/001-short-working-title/
episodes/002-short-working-title/
```

Use three digits so folders sort correctly.

## Episode Files

Use the same core files in each episode folder:

- `brief.md`
- `research.md`
- `outline.md`
- `recording-notes.md`
- `transcript.md`
- `show-notes.md`
- `promotion.md`
- `retrospective.md`

## Asset Names

For external media storage, prefer:

```text
001-working-title_raw-host-audio_2026-06-15.wav
001-working-title_final-mix_v01.mp3
001-working-title_captioned-clip_01.mp4
```

Use lowercase words, hyphens, version numbers, and dates.
